The easier carousel implementation that you can find it.

== Description ==

That is a very basic plugin that allows you to use the carousel functionality in your wordpress website

Major features in Arthur Carousel include:

* Add photos and alts
* Add/remove slide control
* Add/remove fade animation


== Installation ==

Upload the Arthur Carousel plugin to your blog and activate it.
Then go to the "Site Scripts" Tab to configure your plugin.
